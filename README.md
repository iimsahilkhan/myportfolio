# Assets Folder

Place your resume PDF file here as `resume.pdf` to enable the "Download Resume" button in the hero section.

Add your profile photo as `profile_image.jpg` to display it in the hero section.

Your portfolio is now complete and organized with the following structure:
- `/css` - All stylesheets
- `/js` - All JavaScript files
- `/assets` - Images, profile photo, resume, and other assets
